{{/* album shortcode — links to an album page in the photo library, optionally showing the album's cover photo thumbnail. Usage: {{< album slug="france-2024" >}} {{< album slug="france-2024" title="France 2024" >}} {{< album slug="france-2024" title="France 2024" thumb="true" >}} Parameters: slug (required) URL slug of the album (shown in 'photo album list') title (optional) link text; defaults to "View album" thumb (optional) "true" shows the album cover thumbnail above the link The cover thumbnail is served at /albums/{slug}/cover and resolves automatically server-side — no photo ID needed. If the album has no cover photo, or the cover photo isn't published, the image fails to load; pair with onerror handling in your site CSS/JS if you want a placeholder. Set site.Params.photoServer in your Hugo config to your photo server base URL: [params] photoServer = "https://photos.yourdomain.com" */}} {{- $slug := .Get "slug" -}} {{- $title := .Get "title" | default "View album" -}} {{- $thumb := .Get "thumb" | default "false" -}} {{- if not $slug -}} {{ errorf "album shortcode: slug parameter is required" }} {{- end -}} {{- $base := "" -}} {{- with site.Params.photoServer -}} {{- $base = . -}} {{- else -}} {{- $base = "https://photos.example.com" -}} {{- end -}} {{- $albumURL := printf "%s/albums/%s" $base $slug -}} {{- $coverURL := printf "%s/albums/%s/cover" $base $slug -}} {{- if eq $thumb "true" }} {{ $title }} {{- else }} {{ $title }} {{- end }}
